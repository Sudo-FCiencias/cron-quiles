## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad, por favor **NO** crees un issue público.

En su lugar, por favor:

1. Envía un email a los mantenedores del proyecto (o abre un issue privado si es posible)
2. Proporciona detalles sobre la vulnerabilidad
3. Incluye pasos para reproducir el problema
4. Espera una respuesta antes de hacer pública la vulnerabilidad

## Qué Reportar

Por favor reporta:
- Vulnerabilidades de seguridad en el código
- Problemas de privacidad
- Problemas relacionados con el manejo de datos de usuarios
- Cualquier otro problema de seguridad relevante

## Qué NO Reportar

No reportes como vulnerabilidad:
- Problemas de estilo de código
- Sugerencias de features
- Bugs no relacionados con seguridad

## Proceso

1. Recibiremos tu reporte y lo revisaremos
2. Te confirmaremos la recepción del reporte
3. Investigaremos el problema
4. Te mantendremos informado sobre el progreso
5. Una vez resuelto, publicaremos un aviso de seguridad (si es apropiado)

Gracias por ayudar a mantener seguro este proyecto.
